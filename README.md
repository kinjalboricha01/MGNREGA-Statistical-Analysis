# MGNREGA: A Data-Based Comparative Study of Indian States

## Project Overview
This project analyzes the impact and performance of MGNREGA across Indian states using statistical techniques and data-driven insights. The study focuses on employment generation, expenditure patterns, women participation, and state-wise performance indicators.

## Objectives
- Analyze employment trends under MGNREGA.
- Study expenditure and wage distribution patterns.
- Compare the performance of different Indian states.
- Assess the social, economic, and environmental impact of MGNREGA.
